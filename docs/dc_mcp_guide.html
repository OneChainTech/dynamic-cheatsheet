MCP服务

服务遵循 MCP 标准，通过 Server-Sent Events (SSE) 与客户端保持会话。（地址: http://ip:8000/sse）

prepare_solve_context

为业务模型准备上下文。

请求参数
session_id：字符串，会话标识。
返回字段

session_id：原样回传。

cheatsheet：当前会话的记忆内容（若无则为 (empty)）。

update_cheatsheet

利用大模型生成并持久化新的 cheatsheet。

请求参数

session_id：会话标识。

question：原题面/任务描述。

model_output：业务模型输出，需包含 FINAL ANSWER。

返回字段

status：固定为 ok 表示成功。

session_id：会话标识。
